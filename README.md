# hubot-what

A hubot script that does the things

See [`src/what.coffee`](src/what.coffee) for full documentation.

## Installation

In hubot project repo, run:

`npm install hubot-what --save`

Then add **hubot-what** to your `external-scripts.json`:

```json
[
  "hubot-what"
]
```

## Sample Interaction

```
user1>> hubot hello
hubot>> hello!
```
